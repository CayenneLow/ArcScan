# Setup instructions
1) Clone repository
2) ``` npm install```
3) ``` node app.js ``` or ``` nodemon app.js ```(if nodemon installed)

# Troubleshooting: Error with bcrypt.lib
1) ```rm -rf node_modules ```
2) ```npm update```
